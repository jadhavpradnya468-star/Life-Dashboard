#  Life Dashboard 

A modern **productivity dashboard** built using **HTML, CSS, and JavaScript** that combines tasks, habits, and notes into one powerful interface.

---

## 🚀 Features

✅ Task Manager (Add / Delete / Complete tasks)
🔥 Habit Tracker (Track daily streaks)
📝 Notes (Auto-save notes)
🌙 Dark / Light Mode
💾 LocalStorage (data persists after refresh)
⚡ Dynamic UI updates

---

## 🧠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

```
📁 life-dashboard/

```

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/life-dashboard.git
```

2. Open the project:

```bash
cd life-dashboard
```

3. Run:

* Open `index.html` in your browser

---

## ▶️ Usage

1. Add tasks and manage them
2. Track habits with streaks
3. Write notes (auto-saved)
4. Switch sections using sidebar

---

## 💡 How It Works

* Data is stored in `localStorage`
* UI updates dynamically using JavaScript
* Multiple features are combined into one dashboard
